ic-disable-inputs - Disable all inputs when form submit

Summary

Use ic-disable-input when you want to disable all inputs when a request is in process

Example

        <form id="form-on-submit" ic-post-to='/disabled_on_submit'  ic-disable-inputs="true" >
            <input type="text" name="foo" value="bar">
            <input type="submit" value="submit to disabled">
        </form>
